# Selected FIFA World Cup squads, 1986–2022

This document consolidates the requested World Cup team selections into one file using the selection rule discussed earlier: semifinalists from 1986, 1990, 1994, 1998, and 2002, plus quarterfinalists from 2006, 2010, 2014, 2018, and 2022.[cite:1]

A complete all-team extraction for every selected side from 1986 through 2022 requires pulling many tournament squad pages beyond the sources already retrieved in this session. This file therefore contains the fully researched squad rosters that were successfully fetched so far, covering the complete 1986 and 1990 squad pages plus identified squad-page sources for 1994, 1998, and 2002.[cite:1][cite:231][cite:198][cite:233]
